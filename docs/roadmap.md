# Roadmap

## Phase 0 — Baseline and preservation

- freeze the Steam baseline;
- publish cleaned manifests;
- document installation prerequisites;
- record known runtime-generated files;
- define the mod research template.

## Phase 1 — Community archaeology

- recover essential patches;
- recover visual and accessibility mods;
- identify authors and versions;
- record source and permission status;
- diff every archive against vanilla;
- test in isolated copies.

## Phase 2 — Modern compatibility

- dependency diagnostics;
- resolution experiments;
- aspect-ratio handling;
- borderless/fake-fullscreen experiments;
- second-monitor access;
- safe configuration profiles;
- restoration and uninstall workflow.

## Phase 3 — Modern controls

- identify native and hidden shortcuts;
- prototype process-scoped key remapping;
- add ZQSD/WASD camera movement;
- add pause and speed controls;
- document conflict and keyboard-layout handling.

## Phase 4 — Cartographic Edition

- map asset inventory;
- palette and lightmap research;
- art-direction prototypes;
- three-zoom readability tests;
- restoration of all map modes;
- minimap and map-button restoration.

## Phase 5 — Readable Interface

- window and hitbox feasibility tests;
- provisional font metrics;
- campaign-selection vertical slice;
- province/character window vertical slice;
- shared spacing, typography and icon system;
- remaining screens.

## Phase 6 — Onboarding

- campaign recommendations;
- clearer rank and character selection;
- guided tutorial prototype;
- annotated reference material;
- French and English terminology review.

## Phase 7 — Packaging

- modular package format;
- compatibility matrix;
- launcher adapter;
- installer and uninstaller;
- release candidates;
- public documentation.

## Phase 8 — Optional visual expansion

- heraldry;
- portraits;
- event presentation;
- sound and atmosphere.
