# Historical Mod and Patch Inventory

This document tracks candidates reported during preliminary community research. A name appearing here does **not** mean the archive is safe, stable, compatible or redistributable.

## Required metadata

Every recovered item must receive:

- exact name;
- author;
- version;
- release date;
- original source;
- surviving mirror;
- SHA-256;
- modified files;
- dependency and install order;
- gameplay impact;
- save compatibility;
- supported language;
- known bugs;
- permission or licence status;
- test result against the Steam baseline.

## Excluded candidates for the first Golden Build

- total conversions;
- alternate scenarios;
- large Vanilla+ packages;
- map projects that add or remove provinces;
- balance or event expansions that cannot be separated from content changes.

These may be documented as technical references but are not automatically part of the restoration baseline.
