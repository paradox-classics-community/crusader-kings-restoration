# Research Sources

## Purpose

Record every source used to identify or evaluate a patch, mod, tool or engine behaviour. Old forum claims must not be treated as confirmed until an archive or reproducible test is available.

## Source record template

```markdown
## Source title

- URL:
- Archived URL:
- Author:
- Publication date:
- Access date:
- Related item:
- Claims:
- Files or screenshots available:
- Reliability:
- Follow-up required:
```

## Reliability rules

Prefer, in order:

1. original author release post or repository;
2. official Paradox patch notes or preserved forum attachment;
3. source-code repository;
4. archived readme bundled with the mod;
5. independent installation guide with file evidence;
6. community recollection.

Community recollection is useful for discovery but insufficient for integration.
