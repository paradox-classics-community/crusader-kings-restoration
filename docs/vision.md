# Vision

Crusader Kings Complete remains historically interesting, but its presentation and first-use experience prevent many modern players from discovering its strengths.

The restoration aims to create this progression:

1. **Attraction** — screenshots and the map make the game desirable.
2. **Readability** — text, buttons, windows and map modes are clear.
3. **Understanding** — the player understands the available actions and likely consequences.
4. **Comfort** — modern displays, multiple monitors and keyboard controls do not fight the player.
5. **Retention** — a guided introduction turns curiosity into a real campaign.

The project will not redefine CK1 as a modern sequel. It will present the existing game as a carefully restored historical strategy game.
