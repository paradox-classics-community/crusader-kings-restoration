# Art Direction

Art direction work has not started.

The first concepts must satisfy both beauty and strategic readability across:

- three map zoom levels;
- every map mode;
- overlay toggles;
- province selection;
- units, forts and heraldry;
- 1080p, 1440p and 4K output profiles.

Initial CK1 direction under consideration:

- a restrained medieval cartographic identity;
- natural terrain colours;
- matte political colours;
- fine but visible borders;
- decorative detail that recedes during play;
- clear hierarchy instead of visual noise.

No final palette, typography or reference board has been approved.
